Our python program draws a flower in a nature scene. When a bug climbs up and passes through the flower, the flower petals change color!

The Snake survival game is a python game where the player plays as a snake. The main goal of the game is to beat the high score. In order to do so, the player has to move the snake with the arrow keys. If the snake eats the apples it grows bigger and your score goes up. To get to the apples(which are randomly placed) the player has to avoid running into the walls or the snake itself. If you do, then its game over and the game restarts. Overall its a classic game that's fun to play! Can you beat the high score?
